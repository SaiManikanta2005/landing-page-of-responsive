# Responsive Landing Page

A simple and responsive landing page built with HTML5 and CSS3. It includes a header with navigation, a hero section with a call-to-action, and a footer with social links. Designed using Flexbox and media queries for mobile responsiveness.

## Features
- Responsive design
- Flexbox layout
- Hero section with CTA
- Footer with social links

## Tools
- HTML5, CSS3
- Flexbox
- Media Queries
- VS Code + Live Server
